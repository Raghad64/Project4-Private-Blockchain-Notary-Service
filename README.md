# Project #4. Build a Private Blockchain Notary Service

This is Project 4, Build a private blockchain notary service, in this project, I created a Blockchain dataset that allows a star to be stored, a mempool component to store temporal requests and a RESTful API that allow users to interact with the application.  I used LevelDB to be able to persist my blochchain and Express as my Node.js framework.

#### This project depends on the following packages:
- levelDB
- crypto-js
- express
- bitcoinjs-lib
- bitcoinjs-message
- hex2ascii

## Setup project for Review.

To setup the project for review do the following:
1. Download the project.
2. Run command __npm install__ to install the project dependencies.
3. Run command __node app.js__ in the root directory.

The app.js file will start an api server at http://localhost/8000


## 1.POST Request Validation  
This request is to generate a request for validation that takes a wallet address as a json body.

### Request
```javascript
http://localhost/8000/requestValidation
```

The request body should contain an address e.g.: 
```javascript
{ "address":"19xaiMqayaNrn3x7AjV5cU4Mk5f5prRVpL" }
```

### Response 
```javascript
{
    "walletAddress": "19xaiMqayaNrn3x7AjV5cU4Mk5f5prRVpL",
    "requestTimeStamp": "1544451269",
    "message": "19xaiMqayaNrn3x7AjV5cU4Mk5f5prRVpL:1544451269:starRegistry",
    "validationWindow": 300
}
```
`validationWindow` is the time which will decrease until the request is not longer valid or got validated.

## 2.POST Validate Request 
This request is to verify a validation request sent by the previous request (requestValidation) and it takes a wallet address and a signature as a json body.
Note: you will need a a bitcoin wallet to sign a message and then POST the address with the signature.

### Request 
```javascript
http://localhost/8000/message-signature/validate
```
The request body should contain an address and a signature e.g.:
```javascript
{
"address":"19xaiMqayaNrn3x7AjV5cU4Mk5f5prRVpL",
 "signature":"H8K4+1MvyJo9tcr2YN2KejwvX1oqneyCH+fsUL1z1WBdWmswB9bijeFfOfMqK68kQ5RO6ZxhomoXQG3fkLaBl+Q="
}
```

### Response  
```javascript
{
    "registerStar": true,
    "status": {
        "address": "19xaiMqayaNrn3x7AjV5cU4Mk5f5prRVpL",
        "requestTimeStamp": "1544454641",
        "message": "19xaiMqayaNrn3x7AjV5cU4Mk5f5prRVpL:1544454641:starRegistry",
        "validationWindow": 193,
        "messageSignature": "valid"
    }
}
```

## 3.POST Star Data
This request is to submit Star information to be saved in the Blockchain.

### Request 
```javascript
http://localhost/8000/block
```
The request body should contain an address and a Star object e.g.:
```javascript
{
    "address": "19xaiMqayaNrn3x7AjV5cU4Mk5f5prRVpL",
    "star": {
                "dec": "68° 52' 56.9",
```
